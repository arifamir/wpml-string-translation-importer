== Description ==

WPML String Translation Importer is used to import wpml string translations to update their translations.

== Installation ==

1. Upload the widget-clone folder to the /wp-content/plugins/ directory
2. Activate the WP Large CSVs Importer plugin through the 'Plugins' menu in WordPress
3. Navigate to WP Large CSVs Importer on the menu, upload your csv and click on the import button.
